## :rocket: Começando

Esse projeto irá conter os seguintes diagramas para a modelagem dos processos:

- [Caso de Uso](https://en.wikipedia.org/wiki/Use_case_diagram)
- [Hitória de Usuário](https://en.wikipedia.org/wiki/User_story)

Para a implementação dos processos modelados será utilizado a linguagem [Java](https://www.java.com/en/) seguindo todos os princípios de orientação ao objeto junto as boas práticas.

## :thought_balloon: Histórias de Usuário

- Como usuário eu gostaria de poder acessar na plataforma informações sobre meus cursos e minhas atividades para uma experiẽncia mais completa do ambiente acadêmico.

- Eu como secretária gostaria de gerar o currículo com as informações do semestre no sistema para guardar as informações sobre o período letivo, alunos, professores e disciplinas.

- Eu como aluno gostaria de poder realizar minha matricula do semestre pelo sistema para agilizar o cadastro das minahs disciplinas.

- Eu como aluno gostaria de poder cancelar minha matrícula em uma ou várias disciplinas porque caso haja um imprevisto e eu precise de uma maior flexibilidade eu tenha possibilidade de adaptar minha grade de horários

- Eu como aluno gostaria de poder adicionar matéria no meu semestre dentro do período disponível através do sistema para agilizar o processo de manejamento da minha grade horária.

- Como professor eu gostaria de conseguir acessar uma listas dos alunos matriculados nas minhas disciplinas no sistema para agilizar os procedimentos relacionados a chamadas e ao manejamento da turma.

- Eu como usuário gostaria de poder redefinir minha senha pois caso eu esqueça ela, eu ainda possa ter alguma forma de acesso ao sistema.
